# Examples

- `knapsack_sa.m`: 0/1 knapsack (“backpack”) solved with `GEAoptimizer` SA to validate the toolbox run loop, monitoring, callbacks, and result/history wiring.

Run from the repo root (so `+GEAoptimizer` is on the MATLAB path), for example:

```matlab
addpath(pwd)
run("examples/knapsack_sa.m")
```
